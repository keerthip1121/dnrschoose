DonorsChoose

Problem Statement:

DonorsChoose is Kaggle Competition in which we need to predict whether the project proposed by a teacher is accepted or not based on the data given

End Goal:

Our End Goal is to build a model that predicts the approval of the project proposed by the teacher with more accuracy

Data Description:

The following is the Description of data that is used to build our model

project_title Title of the project
project_grade_category Grade level of students for which the project is targeted
project_subject_subcategories One or more(comma-seperated) subject sub categories for the project
project_essay_1 First application essay
project_essay_2 Second application essay
teacher_number_of_previously_posted_projects Number of project applications previously submitted by the same teacher
